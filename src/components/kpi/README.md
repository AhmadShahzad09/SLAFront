### Description

Vue component to make a kpi.

### Install

```sh
npm install --save xxxx/component-kpi

### Slots

| Slot Name | Functionality |
|-----------|---------------|
| number | Block for display a cypher |
| symbol | Block for display a measure symbol |
| title | Block for display a title |
| percentage | Block for display a variation percentage |

### Attributes

| Attribute Name | type | Functionality |
|----------------|------|---------------|
| href | String | Kpi link url |
| arrow | boolean | Print an arrow |
| triangle | boolean | Print a triangle in variation number |
| variation-index | Number | Number concerning the calculation of the variation |
